> Get the appropriate RGB colors for [ANSI color codes][wikipedia]

# ansi-to-rgb

```
npm install ansi-to-rgb
```

## Usage

```js
const rgb = require('ansi-to-rgb');

rgb[0] // => [ 0, 0, 0 ]
rgb[255] // => [ 238, 238, 238 ]
```

---
[wikipedia]: https://en.wikipedia.org/wiki/ANSI_escape_code#8-bit
